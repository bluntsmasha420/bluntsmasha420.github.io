Sixth Post - More CSS Selectors Complete

Tuesday July 29th, 2018

Another big section complete today. I am on a roll. Easier to focus with the little guy visiting the parents.

  • In this section I learned how to add more than one class to a tag simply by adding a space and the next class.
  • Next I learned how to be combine element and class selectors like 'p.info' would be specific to paragraphs with the class info.
  • I learned about getting really specific with elements inside of classes like 'ul.content li strong' would effect the unordered list with the content class and then further only the elements that are in the list and have a strong tag applied.
  • If I need to group two classes to have the same properties all I need to do it add a comma between them like h1, h2 {}. The key to that one is the comma.
  • Pseudo-classes are used to select info that is not part of the webpage structure or can't be expressed using other selectors example: 'a:hover, a:active, a:focus { }' this code will activate when a link or "a" element is acted upon with the conditions hover(mouseOver), active(clicked) or focus(tab selected)
  • Lastly I learned specificity. Which is how your browser selects which attribute to apply if there are conflicting information. It will go from least important element, class then id. The more specific you are like p.class then it will chose the one that has more specificity.
